## Scrapelés
- A sctip két excelt generál egyt a Main sheetel, egyet pedig a Property sheettel.
- Hotcakesbe csak a main sheettel rendelkező excelt lehetett beimportálni.
- A Property sheettel rendelkező excelre rá kell engedni a generate_type_properties.py scriptet.
- A képek megfelelő mappastruktúrába rendezése érdekében le kell futtattni a images_saver.py scriptet.
- ~/Portals/{0}/Hotcakes/Data/Product/ helyre kell beletenni a generált mappákat
- Az SQL adatbázisban meg kell változtatni a StoreId-kat 1 re (Ezekben a táblákban: Product, ProductXCategory, ProductXProperty, stb.)
